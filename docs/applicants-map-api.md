# Карта поступающих

Раздел `/applicants` использует read-only Google Apps Script API и не хранит списки в браузере.

## Сводный список

`GET /exec?action=applicants`

Поддерживаются параметры `university`, `basis`, `confirmation`, `direction`, `priority`, `offset` и `limit`. В общем списке API возвращает только замаскированный код и безопасный `profileKey`.

## Карточка

`GET /exec?action=applicant&profileKey=...`

Точный поиск выполняется через `applicantId`. Полный код показывается только в результате такого поиска, не помещается в адрес страницы и не сохраняется в `localStorage`.

До первого использования в Google Таблице нужно запустить меню `Трекер поступления → Начать сбор карты поступающих`. Сбор выполняется порциями и публикует листы `Все заявления` и `Поступающие` после завершения.
